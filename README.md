# 🚀 PPO-3D-UAV-Path-Planning-Landing

> 基于 PPO (Proximal Policy Optimization) 深度强化学习的三维无人机路径规划、动态避障与平滑着陆算法。
> 本项目构建了一个高度自定义的 `gymnasium` 强化学习环境，支持三维连续空间的局部激光雷达感知与“动态下滑道 (Glide Slope)”着陆机制，内置了工业级标准的 PPO 训练管线以及高质量的 3D 轨迹渲染系统。

## ✨ 核心特性 (Features)

* 🌍 **定制化三维微观环境**：构建了 20m x 20m x 2m 的三维连续飞行空间，支持球体与正方体等复杂静态障碍物的拒绝采样 (Rejection Sampling) 生成，内置严谨的起终点防重叠初始化机制。
* 🛬 **动态下滑道着陆机制**：解决了强化学习中“巡航高度保持”与“目标降落”的奖励冲突。环境能够在距离终点 2m 范围内自动构建平滑下降的虚拟滑道，引导无人机实现无伤的精准着陆。
* 🧠 **PPO 强化学习底座**：采用连续动作空间的 PPO 算法架构。集成了网络参数正交初始化 (Orthogonal Initialization)、独立于状态的动作方差 (State-independent LogStd)、广义优势估计 (GAE)、优势函数归一化以及学习率线性衰减等核心稳定机制。
* 📊 **状态空间全归一化**：对无人机获取的 3D 坐标、速度、相对目标向量与 16 线局部 Lidar 雷达数据进行严格的 [-1.0, 1.0] 归一化处理，杜绝高维多尺度输入导致的梯度爆炸，极大提升网络收敛速度。
* 📈 **工业级渲染与数据分析**：内置完备的 Pandas 数据日志系统，记录单步奖励组件、成功率与末端距离；并提供基于 Matplotlib 的高级 3D GIF 渲染接口，直观展现多视角的动态飞行轨迹与避障过程。

## 🧠 强化学习环境设计 (Environment Design)

本项目针对无人机在三维复杂环境中的“避障”与“着陆”耦合难题进行了解耦。核心的状态空间、动作空间与奖励函数设计如下：

### 1. 状态空间 (Observation Space)

智能体的局部观测状态被设计为 **26 维连续特征向量 (Continuous 26D)**。为了保证泛化性，所有物理量均映射至 [-1.0, 1.0] 之间：

| 维度索引 | 物理含义 (Description) | 数据范围 / 说明 |
| :---: | :--- | :--- |
| `0:3` | **自身归一化坐标**：无人机在 X, Y, Z 三个维度的当前位置。 | 映射至环境边界 [-1.0, 1.0]。 |
| `3:6` | **自身归一化速度**：当前 $V_x, V_y, V_z$。 | 结合最大线速度与垂直速度限幅进行归一化。 |
| `6:9` | **相对目标向量**：目标点相对于无人机的三维方向向量。 | 引导无人机建立全局方向感。 |
| `9:10` | **归一化目标距离**：无人机到终点的欧式距离。 | 截断在 [0, 1.0] 内，用于衡量任务进度。 |
| `10:26` | **局部 Lidar 感知**：16 线一维激光雷达的探测数据。 | 探测水平 360° 障碍物，返回到障碍物表面的归一化距离。 |

### 2. 动作空间 (Action Space)

采用 **3 维连续动作空间 (Continuous 3D)**，底层解算基于三维质点运动学模型：

* **$a_0, a_1$ (水平速度控制)**：归一化输出 [-1.0, 1.0]，环境内映射至无人机的实际水平线速度 $[-V_{max}^{xy}, V_{max}^{xy}]$。
* **$a_2$ (垂直速度控制)**：归一化输出 [-1.0, 1.0]，环境内映射至无人机的实际垂直速度 $[-V_{max}^{z}, V_{max}^{z}]$。

### 3. 奖励函数设计 (Reward Function)

本环境的奖励函数专为“协同多约束”而生，巧妙化解了悬停刷分的局部最优陷阱：

$$R_{step} = r_{prox} + r_{dir} + r_{height} + r_{smooth} + r_{step}$$

* 🟢 **距离逼近奖励 ($r_{prox}$)**：向终点产生实质性位移时给予奖励，并分为远距离引导与近距离强吸入两段系数。
* 🟢 **动态方向引导 ($r_{dir}$)**：将速度方向与目标方向的夹角余弦值作为奖励，并与水平速度大小强绑定，杜绝“原地指向刷分”。
* 🟢 **动态下滑道奖励 ($r_{height}$)**：2m 外巡航时鼓励维持 1.0m 目标高度；2m 内降落时，目标高度随水平距离线性降至 0m，并叠加对高频下降速度 ($V_z$) 的惩罚。
* 🔴 **动作平滑惩罚 ($r_{smooth}$)**：对相邻步动作变化率的平方施加微小惩罚，消除高频抖动，提升物理可执行性。
* 🔴 **时间流逝惩罚 ($r_{step}$)**：每步施加固定的重度惩罚，逼迫智能体全速前往终点，彻底粉碎“悬停”策略。

**🏆 终局截断与稀疏奖励防崩溃设计：**
为了防止连续空间的梯度方差过大导致网络崩溃：
1. **常规步幅严格截断**：将所有连续动作项的组合奖励 $R_{step}$ 严格限幅 (Clip) 在 **[-2.0, 2.0]** 之间。
2. **终局事件直通**：触碰静态障碍物或飞出边界将触发 -200.0 的碰撞极刑并结束回合；完美降落至终点区域（水平误差 < 0.5m 且高度 < 0.1m）给予 +200.0 的成功奖励。此类稀疏信号绕过截断器，提供最清晰的价值导向。确保单轮回报完美控制在 [-400, 400] 范围内。

## 🛠️ 环境依赖 (Requirements)

本项目基于 Python 开发，推荐使用国内清华源极速配置环境：

核心依赖库：
* `torch` (PyTorch >= 2.0)
* `gymnasium`
* `numpy`
* `pandas`
* `matplotlib`
* `pillow` (用于 GIF 渲染)

**快速安装指南 (Conda + Pip)：**
```bash
# 1. 创建并激活虚拟环境
conda create -n uav_ppo python=3.10 -y
conda activate uav_ppo

# 2. 安装科学计算与强化学习基石
conda install numpy pandas matplotlib pillow gymnasium -c conda-forge -y

# 3. 安装 PyTorch (以 CUDA 12.4 为例，推荐使用 pip + 清华源保证速度)
pip install torch torchvision torchaudio --index-url [https://download.pytorch.org/whl/cu124](https://download.pytorch.org/whl/cu124)
```
